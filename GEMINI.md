# sharp-skills for Gemini CLI

You have access to the `sharp-skills` repository, which is a general-purpose marketplace for reusable agent skills.

## What to do

- Inspect `skills/` for available skills
- Read a skill's `SKILL.md` before using it
- Prefer the most specific skill that matches the user's request
- Use `skills-index.json` and `docs/skills-index.md` as the published catalog

## Current published skills

- `presentation-microsite`
- `presentation-stage-app`

The repository is intended to expand beyond its current presentation-focused starting set.
