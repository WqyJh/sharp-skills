# Presentation Microsite Brief Template

Use this template when scoping a new microsite.

## Objective
- What should the viewer believe, understand, or do after this experience?

## Audience
- Who is this for?
- What do they already know?
- What are they skeptical about?

## Delivery context
- live presentation
- async link sharing
- mobile-heavy audience
- desktop-first audience
- booth / kiosk / large display

## Story beats
1. Hook
2. Context
3. Insight
4. Solution
5. Proof
6. Highlight moment
7. CTA

## Media inventory
- logos
- screenshots
- diagrams
- demos
- data
- existing video assets
- Remotion-worthy sequences

## Visual direction
- calm / premium / futuristic / editorial / technical / playful
- key color constraints
- typography constraints
- references the user likes

## Interaction budget
- low: subtle reveal and transitions only
- medium: sticky sections and controlled scroll states
- high: complex scroll choreography, synchronized media, 3D or advanced diagrams

## Delivery outputs
- deployed site
- presenter mode
- exportable videos
- screenshots / social assets
- PDF fallback
