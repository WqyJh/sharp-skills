# Presentation Stage App Brief Template

## Objective
- What is the single conclusion the audience should leave with?

## Presentation mode
- in-person talk
- remote screen share
- recorded walkthrough
- kiosk / looping presentation

## Runtime constraints
- expected screen ratio
- projector or TV
- presenter notes needed?
- keyboard / clicker control needed?
- offline-safe required?

## Deck structure
1. Opening
2. Context
3. Problem
4. Solution
5. Proof
6. Demo
7. Close

## Content inventory
- copy
- charts
- product screenshots
- video clips
- logos / brand assets
- demos / embeds

## Visual direction
- premium / minimal / technical / bold / editorial
- typography constraints
- contrast requirements

## Interaction budget
- low: transitions and reveals only
- medium: selective interactive states
- high: custom presenter view, embedded mini-demos, hybrid video/live scenes

## Outputs
- live presentation app
- presenter mode
- recording mode
- Remotion intro/outro
- PDF snapshot fallback
