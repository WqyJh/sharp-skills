# Install sharp-skills for Codex-like workflows

1. Clone this repository.
2. Choose the skill you want from `skills/`.
3. Copy that skill folder into your agent's supported skills directory, or adapt it to your platform's skill loading mechanism.
4. Start a new session and use a request that should trigger the selected skill.

## Published skills

- `presentation-microsite`
- `presentation-stage-app`

## Notes

- `sharp-skills` is a general-purpose skill marketplace repository.
- The current catalog is small, but the repository is intended to grow beyond its initial presentation-focused skills.
